# Stack Implementation using Array

## Problem Statement
Implement a stack data structure using an array. The stack should support the following operations:
1. `push(value)`: Add an element to the top of the stack.
2. `pop()`: Remove and return the element at the top of the stack.
3. `top()`: Return the element at the top of the stack without removing it.
4. `isEmpty()`: Return true if the stack is empty, otherwise return false.
5. `size()`: Return the number of elements in the stack.
